# FunHub

FunHub is a single-page website combining **mini games, calculators, and a fun section**.  
Built with **HTML, CSS, and JavaScript**, it’s interactive, beginner-friendly, and perfect for showcasing web development skills.

## Features
- 🎮 **Tic-Tac-Toe game**: Play against a friend, includes a reset button
- 🧮 **Simple Calculator**: Add, subtract, multiply, divide numbers
- 😂 **Fun Section**: Random jokes to lighten your mood

## Usage
1. Open `index.html` in any modern browser  
2. Use the navigation menu to jump between sections  
3. Play games, calculate, and enjoy jokes  

## Tech Stack
- **HTML**: Structure of the website  
- **CSS**: Styling and layout  
- **JavaScript**: Interactivity (games, calculator, jokes)
